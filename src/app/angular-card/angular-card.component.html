Angular Project Config Setup and configure angular project

Create New Angular Project

ng new 'project-name'

Install boostrap

npm i boostrap

Install jquery

npm i jquery

Import Bootstrap in src/styles.scss

@import "~bootstrap/dist/css/bootrap.css"

replace all contents from src/app/app.component.html with the following

<router-outlet></router-outlet>

Generate component (Home Component)

ng g c home
